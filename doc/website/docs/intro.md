# 关于

## 项目介绍

旨在不干扰其他功能的情况下创建技能组件，以便于在不同的项目中使用，适配多种场景。

采用多附属的设计 以RaySkillSystem 为中心 拓展出各种功能 形成生态

## 项目特点

- 采用多附属的设计 以RaySkillSystem 为中心 拓展出各种功能 形成生态
- 支持其他插件/脚本进行侵入式开发 提供了@RaySkillSystem标签 可以覆盖大多数默认组件
- 支持多种技能类型 技能效果 实现网游中复杂的技能逻辑
- 支持多种技能效果展示方式 粒子 特效 模型特效 等等 （需自行设计）
